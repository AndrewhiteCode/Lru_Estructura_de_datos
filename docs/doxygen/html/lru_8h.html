Detailed Description

API pública de la caché LRU basada en lista doblemente enlazada + índice A..Z.

Modelo:

  • Lista doblemente enlazada para el orden de recencia: head=MRU, tail=LRU.
  • Índice directo index[26] para acceso O(1) por clave 'A'..'Z'.

Operaciones:

  • lru_create / lru_free: gestión de memoria.
  • lru_add / lru_get: mutan el orden de recencia (O(1)).
  • lru_search / lru_print_all: observación (O(n)).
